GTJ2018使用流程

  2020-02-15   建筑方面   工程造价 0 0  评论 字数统计: 110字 阅读时长: 1分

GTJ2018使用步骤：

  1. 建工程。设置工程的计算规则，室外地坪标高

  2. 建楼层。楼层层高，首层底标高，每一楼层的混凝土标号

  3. 建轴网。上下开间和左右进深。

  4. 绘图输入。定义构件、修改属性、绘制构件、套做法

  5. 查看报表

其它的就是细节的问题了。
