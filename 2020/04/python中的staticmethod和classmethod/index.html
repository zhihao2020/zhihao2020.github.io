Python中的staticmethod和classmethod

  2020-04-26   Python   Python内建函数 0 0  评论 字数统计: 166字 阅读时长: 1分

@classmethod

返回函数的类方法

classmethod修饰符对应的函数不需要实例化，不需要self参数，但第一个参数需要是表示自身类的cls参数，可以调用类的属性，类的方法，实例化对象等。

实例

class test(object):
    bar = 1
    def func1(self):  
        print ('AAA') 
    @classmethod
    def func2(cls):
        print ('func2')
        print (cls.bar)
        cls().func1()  

test.func2()               # 不需要实例化

输出：

func2
1
AAA

@staticmethod

返回函数的静态方法

class test(object):
    @staticmethod
    def fun():
        print('hello');
 
test.fun()      # 静态方法无需实例化
obj = test()
obj.fun()        # 也可以实例化后调用

输出：

hello
hello
