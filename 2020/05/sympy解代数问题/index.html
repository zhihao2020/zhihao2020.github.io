Sympy解代数问题

  2020-05-09   Python   数据分析 0 0  评论 字数统计: 1407字 阅读时长: 3分

我们大多数都是数学运算，其实我们还有另外一种方式——符号数学（symbolic math）。这个需要我们是要第三方库Sympy。

定义符号与运算

符号是构成符号运算的基础。

>>>from sympy import Symbol
>>>x = Symbol('x')

这里创建了一个Symbol类的对象，将’x'作为参数传递。注意’x'是作为有引号包围的字符串传递。

这里建议，你使用一个与被指代符号字母相同的标签。

我们可以多次创建单独的Symbol对象或者使用symbols()函数更简洁地定义。

>>>from sympy import symbols
>>>x,y,z = symbols('x,y,z')

分解与展开表达式

factor()函数分解表达式；expand()函数展开表达式，将表达式表示为单个项目的总和。

>>>from sympy import Symbol
>>>from sympy import factor
>>>x = Symbol('x')
>>>y = Symbol('y')
>>>expr = x**2 - y**2
>>>factor = factor(expr)
(x-y) * (x+y)
>>>expand(factor)
x**2 - y**2

如果你试图分解无法分解的表达式，factor()函数会返回原始表达式。

让表达式整齐输出

我们就要使用pprint()函数，这个函数将使表达式看起来整齐些。

>>>from sympy import pprint
>>>expr = x**2 + 2*x*y + y**2
>>>pprint(expr)

这些项按x的幂顺序排列，从最低次到最高次，这需要init_pprint()函数。

>>>from sympy import init_print
>>>from sympy import pprint
>>>init_print(order='rev-lex')
>>>expr = x**2 + 2*x*y + y**2
>>>pprint(expr)

为符号赋值

使用sub()函数将值带入：

>>>expr = x*x + x*y +y*y
>>>res = expr.subs({x:1,y:2})

我们其实也可以用一个符号表示另一个符号：

>>>expr = x*x + x*y +y*y
>>>expr.subs({x:1-y})

简化结果

如果存在相互抵消的项，我们可以用Sympy的simplify()函数

>>>from sympy import simplify
>>>expr = x*x + x*y +y*y - y*y
>>>simplify(expr)

将字符串转换为数学表达式

将用户输入的（字符串）转换为可以执行的数学运算内容。

sympify()函数就可以完成这个任务。

>>>from sympy import sympify
>>>expr = input('Please enter a mathematical expression:')
>>>expr = sympify(expr)

这里可能会出现用户表达式输入错误的情况。这时候，我们可以从sympy.core.sympfy导入SympifyError异常，然后进行异常处理。

表达式乘法

如果我们想计算两个表达式的成绩并输出结果，那就要使用expand()函数

from sympy import expand,sympify
from sympy.core.sympify import SympifyError

def product(expr1,expr2):
    print(expand(expr1*expr2))
   
if __name__ == "__main__":
    expr1 = input('Enter the first expression:')
    expr2 = input('Enter the second expression:')
    
    try:
        expr1 = sympify(expr1)
        expr2 = sympify(expr2)
    except SympifyError:
        print("Invaild input:")
    else:product(expr1,expr2)

解方程

使用slove()解方程。当你输入一个代表变量（比如x）的符号表达式时，slove()函数可以计算出该符号的值。不过，该函数总是通过假设输入的表达式等于0来计算的。

from sympy import Symbol,solve
x = Symbol('x')
expr = x -5 -7
print(slove(expr))

[12]

解二元方程

from sympy import solve,Symbol
x = Symbol('x')
expr = x**2 + 5*x +4
print(solve(expr,dict=True))

[{x:-4,{x:-1}}]

其中dict=True就是指定结果作为一个列表返回，其中每个元素都是Python字典。

用其他变量求解一个变量

这其实也是都solve(),不过就是多了一个参数。

from sympy import solve,Symbol
x = Symbol('x')
y = Symbol('y')
z = Symbol('z')
expr = a*x*x +b*x +c
print(solve(expr,x,dict=True))

多的这个参数，就是告诉slve()函数，你要去解哪个变量。

解线性方程

例如: $$ 2x+3y=6 $$

$$ 3x+2y=12 $$

我们可以这样做：

from sympy import Symbol,solve
x = Symbol('x')
y = Symbol('y')
expr1 = 2*x+3*y-6
expr2 = 3*x+2*y-12
print(solve((expr1,expr2),dict=True))

[{y:-6/5,x:24/5}]

用SymPy绘图

Sympy绘图其实是调用了matplotlib。

from sympy import Symbol
from sympy.plotting import plot
x = Symbol('x')
plot(2*x+3)

这里x轴的坐标是默认自动选择的。我们可以通过调节plot()函数中的参数，改变x轴的范围。我们也可以改变x轴标签，y轴标签和标题名称

plot((2*x+3),(x,-5,5),title='Line',xlabel='x',ylabel='2x+3')

可以将plot对象调用save()函数将图形保存为图像。

多图像绘制

from sympy import Symbol
from sympy.plotting import plot
x = Symbol('x')
p = plot(2*x+3,3*x+1,legend=True,show=False)
p[0].line_color = 'b'
p[1].line_color = 'r'
p.show()

legend参数用于显示图例。rashow之所以设为不显示，是为了更改线条的颜色。

解单变量不等式

slove_ploly_inequality()函数，用于解多项式不等式。

slove_rational_inequality()函数，用于解有理式不等式。

slove_univariate_inequality()函数，用于解非有理式，非多项式的不等式
